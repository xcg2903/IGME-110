How I created the website

I coded the majority of this website using Notepad on my computer. I had never used html before this class so I didn’t feel like downloading and learning how to use an entirely new program. I also used an online html editor to help me with some of the more difficult pieces of code. If I wanted to test how something would work without having to actually enter it into my project, I could simply write it into the html editor and the live window would immediately tell me how it was going to look. Additionally, I used the website w3schools to find specific pieces of code that I didn’t know how to write myself. For example, there was a page on that website that explained how to create the page bar that I have at the top of all of the information pages. Trying to figure out how to create that on my own would’ve been challenging for a beginner so I’m grateful that I found a website that could show me. I also used the website Paletton to create the grey-blue color palette that I used to create the interface of my site. The actual layout of my website is quite simple. The cover pages has various images that link you to different pages of the website, and once you are on those pages you can navigate between them using the overhead bar. The text layout is formatting using tables, that way you can see all the important text and images right in front of you without having to scroll down (except for the works cited page).
